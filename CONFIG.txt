HOW TO INSTALL DEPENDENCIES:

  WINDOWS:

    - Windows 10 or Windows 11 (64-bit)
    - Python 3.11  →  https://www.python.org/downloads/release/python-3119/
        During install: check "Add Python to PATH"
    - Ollama       →  https://ollama.com/download/windows
        Run the installer, then verify with:  ollama --version(terminal)

    1. Install Python 3.11 and Ollama (links above)
    2. Double-click setup.bat — handles everything else
    3. Pull your chosen models per the tables below
    4. Run:  py -3.11 launch.py

  MacOS:

    - macOS 10.14+ (Mojave or newer)
    - Python 3.11  →  https://www.python.org/downloads/macos/
        During install: check "Add Python to PATH"
    - Ollama       →  https://ollama.com/download/mac
        Run the installer, then verify with:  ollama --version
    - Homebrew     →  https://brew.sh
        Required for system packages. Install it first, then:
        brew install tesseract
        brew install brightness
    - DO NOT install pywin32, pywinauto, or pygetwindow — these are Windows-only
      and will cause pip errors on macOS. They are commented out in requirements.txt.

    1. Install Python 3.11, Ollama, and Homebrew (links above)
    2. Run brew install tesseract and brew install brightness
    3. In Terminal:
           chmod +x setup_mac.sh
           ./setup_mac.sh
    4. Pull your chosen models per the tables below
    5. Run:  python3.11 launch.py

  LINUX(Ubuntu 20.04+ or any modern distro):

    - Python 3.11
        sudo apt install python3.11 python3.11-dev
    - Ollama
        curl -fsSL https://ollama.com/install.sh | sh
        Verify with:  ollama --version
    - System packages (setup_linux.sh installs these automatically):
        sudo apt install tesseract-ocr
        sudo apt install espeak           ← TTS backend for pyttsx3 on Linux
        sudo apt install brightnessctl
        sudo apt install wmctrl           ← needed for WhatsApp window focus
    - Audio: needs PulseAudio (pactl) or PipeWire (wpctl) — usually pre-installed
    - Note: for brightness control without sudo every time:
        sudo usermod -aG video $USER
        (log out and back in after running this)
    - DO NOT install pywin32, pywinauto, or pygetwindow — these are Windows-only
      and will cause pip errors on Linux. They are commented out in requirements.txt.

   1. Install Python 3.11 and Ollama (commands above)
    2. In Terminal:
           chmod +x setup_linux.sh
           ./setup_linux.sh
    3. Pull your chosen models per the tables below
    4. Run:  python3.11 launch.py

this uses two model slots, both configured in config.json:

    "vision_model"    — handles screen analysis, and icon finding, basically anything visual
    "reasoning_model" — handles math, code, and deep reasoning queries and normal conversations

  You only need to set these two keys. Pick one model for each role from the
  tables below based on your hardware

  Ollama stores models in:
    Windows  →  C:\Users\<YourName>\.ollama\models\
    Mac      →  ~/.ollama/models/
    Linux    →  ~/.ollama/models/

HOW TO DOWNLOAD MODELS:

  Make sure Ollama is running (it starts automatically on Windows after install).
  Then open a Command Prompt and pull your chosen models:

  Examples:

    ollama pull moondream:1.8b        
    ollama pull qwen2.5:3b                       
    ollama pull qwen2.5:7b          
 
  To see all models you have installed:
    ollama list

  To remove a model you no longer want:
    ollama rm <model-name>

  Large models can be 5–40 GB each — make sure you have the disk space.

HOW TO TELL PROGRAM WHAT AGENTS YOUVE INSTALLED:
  Open config.json in any text editor and set your model names:

    {
      "vision_model":    "moondream:1.8b",   ← change to your chosen vision model
      "reasoning_model": "qwen2.5:3b",       ← change to your chosen reasoning model
      ...
    }

  *****The model name must exactly match the tag shown by  ollama list.*****
-------------------------------------------------------------------------------------------------
  Other settings you may want to review(optional):
    "wake_word"        — "Jarvis"/"Jarv" by default
    "allowed_actions"  — list of actions the assistant is permitted to take
    "tts_enabled"      — true/false to toggle spoken responses

ISSUES:
  Symptom: ~45-second delay on startup, Silero ONNX warning in logs
  Fix:     pip install silero-vad.

  Symptom: "tesseract is not installed or not in PATH"
  Fix:     Complete Section 2 and restart your terminal.

  Symptom: float16 warning on startup
  Fix:     Harmless. CT2 auto-converts to float32. Ignore it.

  Symptom: DPI awareness warning on startup
  Fix:     Harmless. Ignore it.

 Symptom: WhatsApp send fails or times out
  Fix (Windows): pip install pygetwindow   (then uncomment pygetwindow in requirements.txt)
  Fix (Linux):   sudo apt install wmctrl
  Fix (Mac):     no extra packages needed — uses osascript automatically

  Symptom: Model responses are very slow
  Fix:     You may be running on CPU. Check that your GPU drivers are up to
           date, and consider switching to a smaller model.

  Symptom: "model not found" error when starting
  Fix:     Run  ollama list  and confirm the model name matches exactly what
           is set in config.json, including the tag (e.g. qwen2.5:7b not qwen2.5).
